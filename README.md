## Available Skills

### [Teaching](skills/teaching)
Adaptive AI tutor that teaches any topic through structured pedagogy.

**Key Features:**
- **Knowledge Gathering:** Assesses learner level and gathers domain knowledge from project files, documentation, and the web.
- **Structured Teaching Plans:** Creates detailed, modular plans with learning objectives, prerequisites, and exit criteria.
- **Interactive Teaching:** Uses evidence-based techniques like active recall, spaced repetition, and scaffolding.
- **Progress Tracking:** Maintains a `learning-log.md` to track mastery, regressions, and success metrics across sessions.
- **Real-time Adaptation:** Adjusts difficulty and pacing based on learner performance and engagement.
- **Multi-platform Support:** Includes scripts for initializing sessions on Linux, macOS, and Windows.
